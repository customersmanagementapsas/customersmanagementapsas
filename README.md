## Development server

Run `npm i` and `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

To use address autocomplete feature you need an API key from Google. [How to get API key](https://developers.google.com/maps/documentation/javascript/get-api-key?hl=en#key) (credit card is needed, but you get free â‚¬269.49 for testing purposes).

In app.module.ts replace "YOUR_GOOGLE_API_KEY" with your API key.

In Google Cloud Console enable Places API and increase quota for requests per day (because default is 1).
